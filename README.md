# ♻️ Trợ lý Phân loại Rác thải bằng AI (Garbage Classification)

Dự án này sử dụng mô hình học sâu **ResNet50** (viết bằng PyTorch) để nhận diện và phân loại rác thải thành 6 nhóm: Giấy, Bìa cứng, Thủy tinh, Kim loại, Nhựa, và Rác chung. Giao diện Web được xây dựng bằng **Streamlit**.
